## らくちんに高速化したいのですが numba編
Numbaがあります。


## Numbaとは
Numba公式
https://numba.pydata.org/

Numbaは、Pythonのコードを高速なマシンコードに変換するJITコンパイラで、Pythonのライブラリとして扱うことができます。

ソフトウェアを別途インストールする必要はありません。
パスを通したり、設定ファイルを変更することもありません。
ライブラリを`import`して、関数の戻り値と引数の型を指定するだけです。


## 使える要件
Windows/macOS/Linux

Python 3.6-3.9

今日現在であれば以下の条件が必須です。
```
llvmlite 0.36.*
NumPy >=1.15
```

ここらへんも含めて次の項で説明します。

## インストール
１．Pythonのバージョンを確認
```
bash-3.2$ python --version
Python 3.9.1
bash-3.2$
```

２．`pip`と`setuptools`の更新
```
$ python -m pip install --upgrade pip setuptools
```

３．`Numba`のインストール
```
$ pip install numba
```
これにより、`llvmlite`、`numpy`などが必要に応じてインストールされます。

```
bash-3.2$ pip list | grep llvmlite
llvmlite                 0.40.1
bash-3.2$ pip list | grep numpy
numpy                    1.23.3
bash-3.2$
```
